# Domain Documentation

This directory contains high-level overviews of the main domains in the Laravel/Vue application. Each subfolder summarizes the backend controllers, models, jobs, and services along with the relevant frontend pages, components, services, and stores.

## Domains

-   **agents** – Do's and dont's for ai agents
-   **auth** – User authentication and registration
-   **users** – User management

Use these documents as quick context for AI coding agents working within each domain.
